# RoadStyle

Модуль для **Topomatic Robur** - управление конфигурациями слоёв и стилями отображения автомобильных дорог.

---

## Возможности

| Команда | Описание |
|---|---|
| **Применить конфигурацию** | Быстрое применение сохранённого состояния слоёв ко всем открытым моделям |
| **Импорт слоёв XML** | Массовое применение конфигурации слоёв из XML-файла к выбранным моделям |
| **Экспорт слоёв XML** | Экспорт состояний слоёв активной модели в XML для обмена между проектами |
| **Копирование настроек дорог** | Копирование стилей отображения (поверхность, план, ЦММ) с эталонной трассы на остальные |
| **Перестроить поверхность** | Перестроение проектной поверхности и перерисовка откосов для выбранных трасс |
| **Экспортировать стиль** | Сохранение настроек отображения трассы в файл `.rdprofile` |
| **Импортировать стиль** | Применение профиля настроек к выбранным трассам из файла |
 
---

## Установка

1. Скачай `RoadStyle-1.0.1.tpm` со страницы [Releases](../../releases)
2. Открой **Диспетчер модулей**: `...\Topomatic Robur Road 16.0\TopomaticPackageManager.exe`
3. Выбери скачанный `.tpm` файл и установи
4. Запусти программу

После установки в интерфейсе появится вкладка **Конфигурации** и раздел **Модули → Конфигурации**.

## Разработка

### Сборка из исходников

```powershell
dotnet build
powershell -ExecutionPolicy Bypass -File ".\build-tpm.ps1"
```

### Структура проекта

```
Plugin.cs                  # Основные команды (импорт XML, копирование настроек)
ExportLayersCommand.cs     # Экспорт состояний слоёв
QuickLayerStateCommand.cs  # Быстрое применение конфигурации, О модуле
RoadProfileCommand.cs      # Экспорт/импорт профилей стилей
PluginHost.cs              # Точка входа модуля
RoadStyle.plugin           # Манифест: меню, лента, действия
icons/                     # Иконки команд
build-tpm.ps1              # Скрипт сборки .tpm пакета
```
